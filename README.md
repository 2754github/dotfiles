## 概要
環境構築の共有を容易にするためのリポジトリ。

## 導入
```zsh:ターミナル
$ git clone xxx
```

#### インストールから行う
```zsh:ターミナル
$ sh settings/dotfiles/setup.sh
```

#### シンボリックリンクを貼るだけ
```zsh:ターミナル
$ sh settings/dotfiles/link.sh
```

## ターミナル（zsh）の設定
- ターミナルのフォントを`Ricty for Powerline`に変更
- 背景: 不透明度50%
- フォントサイズ: 18
- フォントカラー: シアン
- ウインドウ: 列160 x 行40
